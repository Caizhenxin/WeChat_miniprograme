# 微光日记 - 迭代日志

## 迭代 0 (初始架构)
- 建立基础微信小程序框架
- 搭建云函数 analyzeDiary 基础代码
- 实现本地日记存储、情绪分析
- 创建四大流派 (CBT/ACT/人本主义/叙事) 学习页面

## 迭代 1 - API 集成 (SiliconFlow)
- 原因：需要 AI 生成差异化流派回应
- 接入 SiliconFlow API，使用 DeepSeek-R1-Distill-Qwen-7B 模型
- 实现 8 秒 API 超时 + 20 秒云函数超时
- 添加本地兜底方案（关键词+模式分析）
- 引入否定词检测，防止「我不高兴」误判为高兴
- 引入程度词（非常、特别）强度缩放

## 迭代 2 - UI/UX 修复
- 原因：用户反馈首页输入框不居中、保存按钮被遮挡、页面状态未重置
- 首页改为双状态设计（输入模式 / 结果模式）
- 输入框使用 flex 垂直居中
- 每次进入页面自动重置状态
- 底部留白 calc(120rpx + safe-area) 防止保存按钮被遮挡
- 新增 onShow 同步学习进度

## 迭代 3 (当前) - 温暖美学 + 交互闭环 + 逻辑清洗
- 原因：Aim.md 定义暖白/纸质美学，但实装为紫色科技风；分析后无微行动交互；存在代码冲突
- 优化内容：

### 3.1 顶部 UI 遮挡修复
- 首页添加 `<navigation-bar>` 组件，原生处理安全区域
- 导航栏颜色改为暖白 #FDF8F2，文字 #2B3A42

### 3.2 交互闭环：流派选择 → 微行动
- 流派卡片改为可点击（`onSelectSchool`）
- 选择流派后显示「已选择」徽标和对应微行动卡片
- 微行动卡片展示：名称、预计时长、步骤、提示
- 数据结构增加 `selectedSchool` / `selectedAction`

### 3.3 色彩方案重构（Aim.md 对齐）
- 背景：`#FDF8F2`（暖白）
- 卡片：`#FFFCF7`（米白）
- 强调：`#D4B595`（陶土褐）/ `#C49A76`（深陶土）
- 边框：`#EADBC6`（旧纸黄）
- 主文字：`#2B3A42`（午夜蓝）
- 次要文字：`#5C4F42`、`#8B7E74`、`#B5A99B`

### 3.4 日记页面清洗
- 删除未定义引用：`analysisResult`、`coreSchools`
- 删除损坏的 Hunyuan API 调用，统一使用云函数 analyzeDiary
- 添加缺失的 `goToSchoolsPage()` 函数
- 导航栏颜色同步为暖白

### 3.5 全局样式对齐
- app.wxss 增加 CSS 变量定义暖色系
- schools 页面对齐暖白主题

## 迭代 4 (当前) - UI 全站统一 + 打卡动画 + 兜底机制
- 原因：个人中心/历史/周报/设置页面配色不统一；情绪周报无打卡浇水动画；日记页分析无兜底
- 优化内容：

### 4.1 全站配色统一（暖白纸质美学）
- **个人中心页**：导航栏 #FDF8F2，用户卡片渐变 #D4B595→#C49A76，卡片白底+旧纸黄边框
- **历史/我的日记页**：背景 #FDF8F2，日记卡片白底+旧纸黄边框，流派标签陶土色
- **情绪周报页**：全页暖白背景，情绪分布条陶土褐渐变，评估卡边框同步暖色
- **设置页**：背景/文字/分割线全部对齐暖色系

### 4.2 情绪周报导航修复
- 「开始记录」按钮从 `navigateTo('/pages/diary/diary')` 改为 `switchTab('/pages/index/index')` — 跳转至暖白主题首页，体验一致

### 4.3 日记页改进
- **兜底机制**：云函数分析失败时自动降级为本地关键词分析（不再只弹toast无响应）
- **打卡浇水动画**：首次打卡时弹出「💧浇水」弹窗，显示连续打卡天数和小树成长阶段，2.5秒自动消失
- **雷达图配色**：统一为陶土褐暖色系
- **全页配色**：同步暖白主题

### 4.4 日记页WXSS重写
- 所有卡片白底+旧纸黄边框
- 按钮渐变 #D4B595→#C49A76
- 文字色系 #2B3A42/#5C4F42/#8B7E74/#B5A99B
